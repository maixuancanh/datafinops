# DataFinOps Test Fixtures

These fixtures are deterministic, synthetic, and safe for local development, CI, screenshots, and demo recording. They contain no customer data, production TxLINE payload, funded wallet, credential, or private signing material.

All files use schema version 1.0 and the fixed instant `2026-08-01T12:00:00.000Z`. The provenance object travels with each fixture so downstream evidence can distinguish synthetic replay from live-read or production state.

The datasets cover pricing, the current subscription, fixture schedules, aggregate usage, hard requirements, a public simulation receipt, verified synthetic entitlement, and an exact synthetic savings close. Values are intentionally small and cannot authorize a real transaction.
